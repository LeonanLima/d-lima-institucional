# Roteiro de Atendimento WhatsApp — Pré-Aprovação MCMV
**D'LIMA Engenharia | Leonan de Lima**

---

## CONTEXTO

O lead chega pelo site com: **nome**, **renda (faixa)** e **situação do terreno**.
A mensagem automática que chega é:

> *"Olá! Meu nome é [NOME]. Vim pelo site da D'LIMA Engenharia e quero fazer uma simulação do Minha Casa Minha Vida. Minha renda familiar é: [FAIXA]. Sobre o terreno: [TERRENO]."*

A partir daí, você conduz a conversa em 4 mensagens para coletar os dados de pré-aprovação.

---

## MENSAGEM 1 — Boas-vindas (enviar logo que receber)

```
Olá, [NOME]! Aqui é o Leonan de Lima, engenheiro civil da D'LIMA Engenharia.

Vi que você quer fazer a simulação do Minha Casa Minha Vida. Ótimo.

Para eu montar sua simulação personalizada, preciso de algumas informações rápidas. São poucas perguntas — leva menos de 3 minutos.

Vamos lá?
```

---

## MENSAGEM 2 — Dados do imóvel

```
Sobre o imóvel que você quer construir ou adquirir:

1. Em qual cidade?
2. Qual o tipo? ( novo / usado / lote / lote + construção )
3. Você já tem ideia do valor? Se sim, quanto?
```

> **Nota:** Se o lead respondeu "não sei como funciona" no campo do terreno, adapte:
> ```
> Vi que você ainda não tem o terreno — sem problema, a gente orienta nisso também.
> Qual cidade você pretende construir?
> ```

---

## MENSAGEM 3 — Dados pessoais

```
Perfeito. Agora algumas informações pessoais para montar o enquadramento correto:

4. Qual seu estado civil?
   ( solteiro / casado / união estável / divorciado / viúvo )

5. Qual sua data de nascimento?
```

---

## MENSAGEM 4 — FGTS

```
Última pergunta:

6. Você possui saldo no FGTS?
   Se sim, você tem ideia de quanto tem disponível?

(O FGTS pode ser usado como entrada — quanto maior o saldo, menor o valor financiado)
```

---

## MENSAGEM 5 — Fechamento e próximo passo

```
Ótimo, [NOME]! Já tenho tudo que preciso para montar sua simulação.

Vou analisar o seu enquadramento e te retorno em até [PRAZO] com:

✔ O valor estimado do subsídio
✔ A parcela aproximada
✔ Os próximos passos para iniciar o processo

Qualquer dúvida que surgir enquanto isso, pode perguntar aqui mesmo.
```

## TRATAMENTO DE OBJEÇÕES COMUNS

**"Não sei o valor do imóvel"**
```
Sem problema. Me diz o tamanho que você quer (em metros quadrados, se souber) ou quantos quartos. Com isso já consigo estimar uma faixa de valor para a simulação.
```

**"Não sei quanto tenho de FGTS"**
```
Tudo bem. Você pode consultar pelo app da Caixa (FGTS) ou no site fgts.caixa.gov.br com seu CPF. Se preferir, segue assim mesmo e a gente verifica durante o processo.
```

**"Posso levar meu cônjuge junto?"**
```
Pode sim. Na composição de renda com cônjuge, o enquadramento pode mudar para melhor. Me passa a renda bruta dos dois somada e o estado civil dele(a) também.
```

**"Quanto tempo leva tudo isso?"**
```
Da simulação até a aprovação do crédito na Caixa, em média de 60 a 90 dias. A obra entra depois da aprovação. O prazo da obra depende do tamanho do projeto — defino isso em contrato antes de começar.
```

**"Tenho nome sujo"**
```
Nome sujo é uma das maiores barreiras para o MCMV. Me conta a situação e vejo o que é possível fazer — às vezes tem alternativa, às vezes precisa regularizar primeiro. Melhor saber agora do que lá na frente.
```

---

## DICAS DE USO

- **Não envie tudo de uma vez.** Espere a resposta de cada mensagem antes de enviar a próxima.
- **Use o nome do lead** em todas as mensagens. Personalização gera vínculo.
- **Não use emojis em excesso.** Um por mensagem no máximo. Você é engenheiro, não influencer.
- **Seja rápido no retorno.** Lead que espera mais de 1h no WhatsApp esfria. Responda o quanto antes.
- **Salve os dados num lugar fixo** (planilha, CRM ou caderno) para não perder nenhuma informação entre conversas.
