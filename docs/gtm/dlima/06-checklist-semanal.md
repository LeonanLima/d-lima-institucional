# Ritual de segunda — D'LIMA (20 minutos, toda semana)

*Segunda-feira, 18h–18h20, antes de qualquer tarefa. O ritual existe para UMA coisa: decidir a mudança da semana com número, não com sensação. [IA] pode compilar os números antes; a decisão é sua.*

## Passo 1 · Preencher os 7 números (5 min)

| # | Indicador | Semana passada | Acumulado | Meta 90 dias |
|---|---|---|---|---|
| 1 | Conversas novas no WhatsApp | | | ≥ 40 |
| 2 | % qualificadas (terreno + renda ok) | | | ≥ 35% |
| 3 | Pré-aprovações INICIADAS | | | ≥ 10 |
| 4 | Pré-aprovações APROVADAS | | | ≥ 6 |
| 5 | Reuniões / visitas a terreno | | | ≥ 5 |
| 6 | Degraus de entrada vendidos (R$) | | | ≥ 1 |
| 7 | Contratos de obra assinados | | | 1 |

E os 3 números de esforço (sem eles os de resultado não significam nada):

| Esforço | Semana passada | Acumulado (ciclo) |
|---|---|---|
| Ativações de rede enviadas | | /50 |
| Posts publicados | | /12 |
| Parceiros abordados | | /10 |

## Passo 2 · As 3 perguntas (10 min)

1. **De onde vieram as conversas qualificadas?** (rede / Instagram / parceiro / Google) → o canal que gerou ganha mais esforço na semana; o que não gerou NÃO ganha esforço novo — ganha diagnóstico.
2. **Onde o funil travou?** Olhar etapa por etapa: muita conversa e pouca qualificada = público errado (mensagem/canal); qualificada sem pré-aprovação iniciada = você não pediu os documentos na mesma conversa (disciplina, não estratégia); pré-aprovação sem sair = fila da Caixa (ligar pro CCA, não esperar).
3. **Alguma objeção apareceu 2+ vezes?** → vira post educativo do mês seguinte e/ou ajuste no roteiro de qualificação.

## Passo 3 · UMA mudança (5 min)

Escrever em uma frase: **"Esta semana eu mudo ___ porque o número ___ mostrou ___."**

Uma mudança por semana. Duas mudanças ao mesmo tempo = impossível saber qual funcionou. Se está tudo verde, a mudança é "nada — repetir a dose".

## Gatilhos automáticos (não são decisão, são regra)

| Situação | Ação obrigatória |
|---|---|
| **Dia 30: < 5 conversas qualificadas** com 50 ativações + 12 posts + 8 parcerias feitos | **PARAR tudo.** O problema é posicionamento/mensagem, não esforço. Revisar Seções 2–3 da estratégia antes de gastar mais um dia |
| Dia 30 sem o esforço completo (< 50 ativações etc.) | O teste NÃO falhou — não foi feito. Completar o esforço antes de mudar qualquer coisa |
| Lead quente há 72h sem lista de documentos enviada | Enviar HOJE (template de cobrança no `05`) |
| Pré-aprovação parada > 4 semanas na Caixa | Ligar pro CCA nesta semana |
| 0 indicações de parceiros no dia 45 | Refazer proposta de valor do parceiro (% em vez de fixo) ou matar o canal |
| Semana com < 4h de execução real | Não compensar virando noite: cortar Canal 3 da semana e proteger janelas + rede |
| "Cadê obra sua?" apareceu 3× | Antecipar "Obra Aberta": documentar reforma/obra de conhecido já |

## Registro (histórico das semanas)

Copiar e preencher a cada segunda — 3 linhas bastam:

```
## Semana {n} ({data})
Números: {os que mudaram}
Mudança da semana: {a frase do Passo 3}
Nota: {1 aprendizado ou objeção nova}
```

*Regra de ouro do ritual: número ruim na mesa é melhor que número bonito imaginado. O ritual que você pula na semana boa é o que te salvaria na ruim.*
